Creating Model Variants

Overview

Create custom variants of your models by adjusting their behavior through temperature and system messages.

Temperature

Temperature controls the randomness of the model's responses:

  • 0.0 - Most deterministic, least creative
  • 0.7 - Balanced between creativity and determinism (default)
  • 2.0 - Most creative, most random

System Message

The system message defines the model's behavior and personality. Use it to:

  • Set the model's role (e.g., "You are a helpful assistant")
  • Define personality traits
  • Set specific behaviors or rules

Best Practices

  • Start with the default temperature (0.7) and adjust as needed
  • Be specific in your system messages
  • Test different variants to find what works best
